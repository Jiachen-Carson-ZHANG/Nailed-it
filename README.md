# Nailed It 💅

**AI 驱动的美甲沙龙智能运营平台** — 虚拟试戴 · 智能报价 · 一键预约 · 款式管理

---

## 这是什么

Nailed It 是一个 B2B2C 美甲预约平台，核心目标是消除"灵感 → 预约"之间的摩擦。

- **用户侧**：浏览/上传美甲图片 → AI 识别款式 → 虚拟试戴 → 查看报价 → 直接预约，全程无需人工客服介入。
- **商家侧**：管理价目表（积木拆分模型）、日历、客户消息，完成订单后上传图片即可自动入库扩充款式库。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **AI 款式识别** | 上传美甲图，自动提取甲形、延甲、手绘、花型等属性 |
| **虚拟试戴** | 将目标款式合成至用户手部照片，辅助决策 |
| **智能报价** | 基于"底层 + 加项"积木模型，匹配商家价目表自动出价 |
| **智能预约** | 根据款式制作时长自动锁定美甲师档期，用户一键完成预约 |
| **款式库管理** | 订单完成后自动打标入库，支持商家分类展示与营销推送 |

---

## 用户角色

- **美甲商家**：注册账号，维护价目表、日历与款式库
- **终端用户**：浏览款式、虚拟试戴、在线预约

---

## 本地启动

```bash
# 安装依赖
npm install

# 启动开发服务器
./dev
```

> 需要配置 Supabase 与 AI API 密钥，详见 `.env.example`。

---

## 技术栈

- **框架**：Next.js 15 (App Router)
- **数据库**：Supabase (PostgreSQL)
- **AI**：火山引擎 Ark（Doubao 视觉/图像模型，用于款式识别、虚拟试戴、趋势分析）；OpenRouter / Qwen（文本类辅助流程）
